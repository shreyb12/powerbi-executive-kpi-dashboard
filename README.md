# Executive KPI Dashboard (Power BI)

## Overview
An executive-ready Power BI dashboard designed to track commercial and operational performance through a clean KPI framework, drilldowns, and variance analysis.

## What I built
- KPI framework (Revenue / Margin / Volume / On-time / Utilization – adaptable)
- Executive summary page + drill-through detail pages
- Variance analysis (Actual vs Target, MoM/YoY)
- Role-based views & performance-focused design patterns

## Tech
- Power BI, DAX, Power Query
- Star schema semantic model (facts + dimensions)
- Performance optimization (measure design, model hygiene)
